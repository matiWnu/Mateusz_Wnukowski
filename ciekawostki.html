Ciekawostki o C++:

  • C++ dalej dostaje aktualizacje
  • Do C++ można wstawiać kod bezpośrednio z Assembly i z C
  • Nazwa C++ wzięła się od "C" i "++", ponieważ C++ to C + 1 (a ++ to znaczy że zwiększamy coś o 1)
  • Pierwsza komercyjna wersja C++ pojawiła się w 1985 roku
  • C++ ma jedne z najbardziej skąplikowanych błędów kompilatora
